## Hello!

My name is Nick and I currently work as a Research Professional at the University of Chicago. 
At work, leverage machine learning techniques and causal inference to extract insight from large, complex datasets.
In my free time, I look for any excuse to play with new datasets or make maps. 

My GitHub showcases my work on two projects: one personal and one professional. 
Much of my programming and documentation lives on secure servers, so these repositories represent a subset of my output:
* [discrimination-lawsuits-public](https://github.com/martensn/discrimination-lawsuits-public) showcases my contributions towards a project with one colleague to construct a scalable, replicable pipeline to download dockets and documents from federal employment discrimination lawsuits. Languages: Bash, Python 
* [gaydar](https://github.com/martensn/gaydar) combines data from the Household Pulse Survey and American Community Survey to estimate the spatial distribution of the LGBTQ population with unprecedented granularity. Language: R

The remaining [public repository](https://github.com/martensn/working-paper) hosts the latest draft of my other side project, which uses Revelio resume data to explore when graduates decide to return to their hometown.
Check it out if you want to see how I visualize data, write about research, or just want to learn about local labor markets!
